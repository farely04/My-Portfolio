PROFILE

สวัสดีค่ะชื่อ วรดา ศรีสอาดพันธ์วงษ์ ชื่อ แฟรี่ ค่ะ เรียกว่า"แฟร์"ได้นะคะ^^ อายุ 22 ปี
เรียนอยู่ที่มหาวิทยาลัยธุรกิจบัฒฑิตย์ เรียนที่วิทยาลัยครีเอทีฟดีไซน์แอนด์เอนเตอร์เทนเมนต์เทคโนโลยี หลักสูตรการพัฒนาสื่อเชิงโต้ตอบและพัฒนาเกม
ผลงานต่างๆของแฟรี่จะมีผลงานหลักๆเป็นผลงานเกี่ยวกับด้านอารต์และผลงานเกี่ยวกับการสร้างเกม

ศักยภาพ

ความสามารถหลักของแฟรี่คือการด้านการวาดรูปรวมถึงการออกแบบทัศนศิลป์
ความสามารถรองที่อยู่ในระดับปานกลางคือการเขียนโปรแกรมภาษาภาษา HTML, CSS และ JavaScript รวมถึงภาษาc# เป็นภาษาที่อยู่ระดับพื้นฐาน

เป้าหมาย

เป้าหมายของแฟรี่ต้องการจะเรียนรู้เกี่ยวกับด้านภาษาของโปรแกรมมากขึ้นในอนาคตเพื่อให้เป็นความสามารถหลักแทนการออกแบบทัศนศิลป์
และความสามารถออกแบบทัศน์ศิลป์นั้นเป็นทักษะเสริมที่รองลงมา

จุดแข็งและจุดด้อย

ในด้านของ Soft skill แฟรี่มีการเรียนรู้อย่างเป็นระบบอย่างรวดเร็ว แก้ไขปัญหาเฉพาะหน้าได้ อดทนต่อแรงกดดันได้ดี
มีทักษะของการแก้ไขปัญหา สามารถสื่อสารเป็นทีมได้เช่นการแบ่งหน้าที่และขอความร่วมมือ
ในด้านของจุดด้อย แฟรี่ค่อนข้างลงมือทำเร็วเนื่องจากการเรียนรู้เร็วของแฟรี่ ทำให้ในบางครั้งอาจเกิดข้อผิดพลาดของงานที่ไม่ละเอียด

วิธีการทำงานและงานอดิเรก

วิธีคิดในการทำงาน แฟรี่จะให้ความสำคัญของงานจากรายละเอียดและรวมถึงเดดไลน์ของงาน จากนั้นแฟรี่จะจัดลำดับความสำคัญออกมาให้เป็นระบบในการทำงาน
งานอดิเรกอื่นๆ แฟรี่ชอบฝึกอ่านภาษาอังกฤษจากแอปพลิเคชัน CHARACTER.AI และชอบอ่านการ์ตูนเกาหลีที่เรียกว่า"มังฮวา" เป็นการฝึกภาษาเกาหลี รวมถึงการดูซีรี่ย์ภาษาเกาหลี

แฟรี่หวังว่าในอนาคตแฟรี่จะสามารถบรรลุเป้าหมายให้ได้ตามที่วางแผนไว้
และสามารถสร้างผลงานให้มีคุณภาพให้ดียิ่งขึ้นในอนาคต ยินดีที่ได้รู้จักนะคะ^^
